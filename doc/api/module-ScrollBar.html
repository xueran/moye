<private> options :Object

控件配置项
Type:
  • Object
Properties:
Name Type Description
options.disabled boolean 是否禁用组件
options.main string | HTMLElement 主元素
options.panel string | HTMLElement 需要滚动的元素， 如果不设则按class规则查找`options.prefix` + `panel`
options.thumb string | HTMLElement 滚动条按钮元素 如果不设则按class规则查找`options.prefix` + `thumb` ， 并且将thumb父级元素作为track
options.wheelspeed Number 滚动速度，百分比，越大滚动越快
options.direction string 滚动方向
options.prefix string class默认前缀
options.mode string 使用的模式(`scroll` or `position`)， scrollTop模式or style.top模式 使用的css是不一样的,默认scrollTop模式
options.preventWheelScroll boolean 如果true会始终阻止滚轮滚动页面，及时当前面板已经滚动到头 此处会防止因面板较短导致的页面频繁滚动
options.autoThumbSize boolean 是否自动调整thumb的高度，以适应滚动内容的大小
options.minThumbSize boolean 自动调整thumb高度时，最小的thumb高度(px)
