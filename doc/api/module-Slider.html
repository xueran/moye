Name Type Argument Description
options.main string | HTMLElement 控件渲染容器
options.stage HTMLElement <optional>
 控件动画容器， 如果不设则按class规则查找`options.prefix` + `stage`
options.prevElement HTMLElement <optional>
 prev按钮的容器， 如果不设则按class规则查找`options.prefix` + `prev`
options.nextElement HTMLElement <optional>
 next按钮的容器， 如果不设则按class规则查找`options.prefix` + `next`
options.indexElment HTMLElement <optional>
 轮播索引按钮的容器， 会将第一级子元素设为索引元素， 如果不设则按class规则查找`options.prefix` + `index`
options.auto boolean 是否自动轮播
options.circle boolean 是否播放到结尾时回到起始， 在自动轮播下，需要设置为true
options.autoInterval Number 自动切换动画的延迟时间
options.switchDelay Number 点击切换索引的延迟时间
options.onChange function 当播放索引改变时的事件
options.prefix string 控件class前缀，同时将作为main的class之一
options.anim string 使用的轮播动画， 默认提供`no`,`slide`,`opacity`
options.animOptions Object 轮播动画选项， 不同的动画效果配置可能不一样
